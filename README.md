# Student Result Management System

A comprehensive mini-project designed to streamline the process of managing, storing, and retrieving student examination results. This system provides a secure environment for administrators to manage academic records and an accessible portal for students to view their scores.

---

## 🚀 Features

### Admin Dashboard
* **Secure Login:** Dedicated login portal for administrators and faculty.
* **Student Management:** Add, update, or delete student profiles and registration details.
* **Result Management:** Insert, modify, and publish marks for different subjects and semesters.
* **Class & Subject Management:** Create and organize classes, sections, and corresponding subjects.

### Student Portal
* **Instant Result Retrieval:** Search and view detailed marksheets instantly using a unique Roll Number / Student ID.
* **Clean Interface:** Clear display of individual subject scores, total marks, percentage, and pass/fail status.
